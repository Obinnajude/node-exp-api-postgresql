Capstone Project Api

Below are the endpoints for the api:

  1. POST /api/v1/auth/create-user
  2. POST /api/v1/auth/signin
  3. POST /api/v1/gifs
  4. POST /api/v1/articles
  5. PATCH /api/v1/articles/<:articleId>
  6. DELETE /api/v1/articles/<:articleId>
  7. DELETE /api/v1/gifs/<:gifId>
  8. POST /api/v1/articles/<:articleId>/comment
  9. POST /api/v1/gifs/<:gifId>/comment
  10. GET /api/v1/feed
  11. GET /api/v1/articles/<:articleId>
  12. GET /api/v1/gifs/<:gifId>

To use the api to post article, gifs and comments, you need some authentication. remember only admin can create user
